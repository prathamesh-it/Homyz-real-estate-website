# 🏡 Homyz - Real Estate Web Application

A modern and responsive real estate web application built using React. This project focuses on delivering a clean UI, smooth animations, and an engaging user experience for browsing properties.

---

## 🚀 Features

* Responsive UI with modern design
* Dynamic property slider
* Smooth animations for better UX
* Interactive accordion for content display
* Mobile-friendly navigation menu
* Reusable component-based architecture

---

## 🛠️ Tech Stack

* React.js
* CSS (Custom + Utility Classes)
* Swiper.js
* Framer Motion
* React Icons

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone <your-repo-link>
```

2. Navigate to the project

```
cd homyz
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```
